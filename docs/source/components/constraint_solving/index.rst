Constraint Solving
==================

This section covers the constraint solving components available in the Lotus framework.

.. toctree::
   :maxdepth: 1

   smt
   bdd
   wpds

Lotus provides several constraint solving implementations with different features and capabilities:

- :doc:`SMT Solver <smt>`: Z3 integration for Satisfiability Modulo Theories
- :doc:`BDD <bdd>`: Binary Decision Diagram implementation using CUDD
- :doc:`WPDS <wpds>`: Weighted Pushdown System library
