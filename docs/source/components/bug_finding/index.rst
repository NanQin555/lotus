Bug Finding
===========

This section covers the bug finding components in Lotus.

.. toctree::
   :maxdepth: 1

   kint
   esss

Lotus provides several bug finding components with different capabilities:

- :doc:`Kint <kint>`: A static analyzer for detecting integer-related bugs
- :doc:`ESSS <esss>`: A tool for finding error checking bugs

These bug finding components can be used to detect various types of bugs in programs written in C and C++.
